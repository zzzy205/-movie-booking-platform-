# 网络切换脚本使用说明

## 📁 脚本文件
- `to-out.sh` - 切换到外网环境
- `to-in.sh` - 切换回内网环境  
- `net-status.sh` - 查看当前网络状态

## 🚀 使用方法

### 1. 切换到外网（用于更新）
```bash
# 1. 拔掉内网网线
# 2. 执行脚本
./to-out.sh
# 3. 插上外网网线
# 4. 使用PuTTY连接进行更新
```

### 2. 切换回内网（更新完成后）
```bash
# 1. 拔掉外网网线
# 2. 执行脚本
./to-in.sh
# 3. 插上内网网线
# 4. 验证内网服务
```

### 3. 查看网络状态
```bash
./net-status.sh
```

## ⚠️ 注意事项
- 切换网络前请确保已备份重要数据
- 外网切换会自动备份内网配置
- 如果切换失败，可以运行 `./to-in.sh` 恢复
- 建议在业务低峰期进行网络切换

## 🔧 故障排除
如果网络切换失败：
1. 检查网线连接
2. 运行 `./net-status.sh` 查看状态
3. 尝试手动恢复网络配置
4. 联系技术支持

## 📋 完整操作流程
1. **准备阶段**：U盘传输脚本到服务器
2. **外网切换**：拔内网网线 → 执行 `./to-out.sh` → 插外网网线
3. **更新阶段**：使用PuTTY连接，传输更新文件
4. **内网恢复**：拔外网网线 → 执行 `./to-in.sh` → 插内网网线
